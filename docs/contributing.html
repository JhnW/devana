Contributing guideÂ¶

If you do not know what aspect of the project you want to help, please check Roadmap or write e-mail to me (check the owner of the GitHub repository).

To start contributing Devana you need to checkout project repository. The primary development platform is Ubuntu, but all the tools should work on other systems as well. The development environment only requires a modern version of python (3.6 and above, 3.8 recommended). All the necessary packages are in the requirements.txt file. Repository contains various python helpers scripts names whose name starts with of build_.

We are using gitchangelog commit format to generate change log. Please check gitchangelog.rc file.
